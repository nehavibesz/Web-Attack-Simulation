# Web-Attack-Simulation
Cybersecurity lab demonstrating web attack simulations and traffic analysis using DVWA, Wireshark, and Burp Suite.

## Overview
This project demonstrates web application attack simulations in a controlled lab environment using DVWA (Damn Vulnerable Web Application).

## Tools Used
- Kali Linux
- DVWA
- Wireshark
- Burp Suite
- Firefox

## Attacks Demonstrated
- SQL Injection
- Brute Force Attack

## Learning Outcomes
- HTTP packet analysis
- Web vulnerability testing
- Response analysis
- Authentication bypass concepts

## Disclaimer
This project was conducted in a controlled educational environment for learning purposes only.
